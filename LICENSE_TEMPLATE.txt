LICENSE (choose one)
====================

Pick a license, then replace this file with the real LICENSE text.

Common choices:
- MIT (simple, permissive)
- Apache-2.0 (permissive + patent grant)
- GPL-3.0 (copyleft)
- Unlicense / CC0 (public domain style)

If you want "others can use it but must keep attribution", MIT is usually fine.
If you want a patent grant, pick Apache-2.0.
